# Block Breaker

Block breaker game from a **GameDev.tv** course

### Instructions to download and play

1. Click `Game` folder
2. Click `Block Breaker.zip`
3. Select `Download`
4. Unzip file
5. In unzipped folder doubleclick `Block Breaker.exe`
6. Select Screen resolution: 1400x1050
7. Enjoy!
